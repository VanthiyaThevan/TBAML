# TBAML Frontend

React + TypeScript frontend for the Trade-Based Anti-Money Laundering System.

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Environment Variables

Create a `.env` file:

```
VITE_API_BASE_URL=http://localhost:8000
```

## Features

- ✅ React 18 + TypeScript
- ✅ Tailwind CSS for styling
- ✅ React Query for data fetching
- ✅ UC1 Input Form with validation
- ✅ Results Display with Activity Indicator
- ✅ Flag Display with color coding
- ✅ Source Citation
- ✅ AI Response View
- ✅ Timeline Visualization
- ✅ Loading states and error handling
- ✅ Responsive design

## Development

The frontend runs on port 3000 by default and proxies API requests to `http://localhost:8000`.

Make sure the backend API is running before starting the frontend.
